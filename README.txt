Flask Stack:
1、Bootstrap 3 and Font Awesome 4 with starter templates
2、Flask-SQLAlchemy with basic User model
3、Easy database migrations with Flask-Migrate
4、Flask-WTForms with login and registration forms
5、Flask-Login for authentication
6、Flask-Bcrypt for password hashing
7、Procfile for deploying to a PaaS (e.g. Heroku)
8、pytest and Factory-Boy for testing (example tests included)
9、A simple manage.py script.
10、CSS and JS minification using Flask-Assets
11、Optional bower support for frontend package management
12、Caching using Flask-Cache
13、Useful debug toolbar
14、Utilizes best practices: Blueprints and Application Factory patterns



link: https://pythonspot.com/flask-boilerplate-with-bootstrap-sqlalchemy/
